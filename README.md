## Getting Started

1. Clone the repo:
   ```
   git clone https://github.com/YOUR_USERNAME/technocore-site.git
   cd technocore-site
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the dev server:
   ```
   npm run dev
   ```

4. Build for production:
   ```
   npm run build
   npm run start
   ```
